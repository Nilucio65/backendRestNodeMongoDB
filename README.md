## 💡Dicas
- Clone o projeto
- Renomeie o arquivo .env-example para .env e informe a sua string de conexão ao MongoDB
- Instale as dependências com ```npm i```
- Abra o Terminal no VSCode e informe ```npm run dev```

## 📦Packages Utilizados
```
npm i express
npm i mongodb@4.2
npm i dotenv
npm i nodemon --dev
npm i express-validator
```
